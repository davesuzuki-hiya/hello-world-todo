# TODO_

A minimalist todo app with a Teenage Engineering-inspired design.

## Live Demo

**[Try it here](https://davesuzuki-hiya.github.io/hello-world-todo/)**

## Features

- Create, edit, and delete tasks
- Mark tasks as complete with a satisfying confetti celebration
- Timestamps show when each task was created
- Clean, monospace aesthetic with dark theme
- No backend required - runs entirely in the browser

## Screenshot

The app features a dark graphite color scheme with orange and teal accents, monospace typography, and a minimal interface.

## Usage

1. Type a task in the input field
2. Press Enter or click ADD
3. Click the checkbox to mark a task complete (and enjoy the confetti!)
